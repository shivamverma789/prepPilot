# PrepPilot - Product Overview

PrepPilot is a student-focused placement preparation platform. It helps college students get job-ready by providing:

- **Personalized roadmaps** generated by AI based on the student's skills, goals, and academic background
- **Skill assessments** (quizzes) to evaluate and track proficiency
- **Resume analysis and building** — upload a resume for AI feedback, or build one from scratch
- **Activity tracking** — aggregates coding activity from platforms like LeetCode, Codeforces, CodeChef, HackerRank, and GitHub
- **Interview preparation** — AI-driven mock interview setup
- **Onboarding flow** — deep profiling of the user (career goals, skills, academic context) to personalize all features

The target user is a college student preparing for campus placements or tech job applications.
